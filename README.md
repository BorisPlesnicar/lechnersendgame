# Lechner's Endgame Trainer

Simple English–German vocabulary practice in the browser: **flashcards**, **type the translation**, and **short sentence writing** (both directions). Built with Next.js (App Router), TypeScript, and Tailwind CSS.

## Install & run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy (Vercel)

Import the repo as a Next.js project. No environment variables required.

## Data

Words and example sentences live in `src/data/vocabulary.ts`. Theme preference is stored in `localStorage`.
